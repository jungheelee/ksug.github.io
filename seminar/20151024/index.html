Modern Java web application with Spring

웹기술은 매우 빠른 속도로 변화하고 있습니다. 이 웹기술을 다루는 프론트엔드 영역은 그야말로 춘추전국시대에 견줄만큼 수많은 기술들이 쏟아져 나와 치열한 생존경쟁을 벌이고 있습니다.
시대적 흐름에 맞추어 스프링 진영에서도 웹과 관련된 다양한 기능들을 선보이고 있습니다.
이런 기술의 홍수속에서 눈여겨볼만한 웹기술과 이와 관련된 스프링의 웹기술을 살펴볼 수 있는 자리를 마련했습니다.

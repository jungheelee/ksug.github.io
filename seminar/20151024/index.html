Modern Web Application for Spring

최근 프론트엔드 영역은 춘추전국시대라고 불러도 될만큼 정말 많은 기술들이 쏟아져 나오고 있고, 치열하게 서로 경쟁을 하고 있는듯 합니다.
더불어 스프링 진영에서도 웹 모듈에 정말 다양한 기능들이 추가가 되었고, 뷰 템플릿 엔진을 이용해 직접 구성하는 방식부터 RESTful API를 통해 데이터를 제공하는 등의 다양한 웹 애플리케이션이 개발되고 있습니다.
이런 기술의 홍수 속에 눈여볼만한 웹 기술 또는 스프링의 기능들을 살펴보는 자리를 만들고자 합니다.

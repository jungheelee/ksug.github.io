Modern Data Access for Enterprise Java

최근 엔터프라이즈 자바 애플리케이션에서 데이터를 다루는 기술은 나날이 복잡해지고 있습니다.
대표적인 데이터 관련 기술을 나열해 보자면 NoSQL, 하둡, 클라우드 기반 데이터 서비스 그리고 가장 많이 사용되고 있을 관계형 데이터베이스가 있을것입니다.
자바가 진화하면서 함께 이런 데이터 접근 기술들도 점차 진화화고 있고, 그 복잡성도 증가하고 있습니다.
이 세미나를 통해 현대적인 자바 데이터 접근 기술들 중 JPA(Java Persistence API)와 함께 스프링의 데이터 접근 기술의 핵심으로 자리 잡기 시작한 Spring Data Project를 알아보는 시간을 만들고자 합니다.
